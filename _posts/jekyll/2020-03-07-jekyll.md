# Jekyll 教程

## jekyll 安装

> 安装jekyll 首先需要 ruby 的环境， Macos系统自待ruby环境，不需要安装；Linux则需要安装


- 安装rvm

  [Ruby Version Manager (RVM)][3]

- 安装rvm

```
   rvm install [version]
```

- 安装 jekyll

```

gem install jekyll bundler

```
